# Crimson Cup Bangladesh — Website UI Redesign

Usability-focused redesign of [Crimson Cup Bangladesh](https://crimsoncupbangladesh.com/)'s website, including a Nielsen heuristic evaluation, mood board, style guide, and responsive high-fidelity mockups for desktop and mobile.

## 📌 Project Overview

This project redesigns the user interface of an existing website to improve usability. It follows a full UX process — from identifying real usability issues on the live site to producing high-fidelity, responsive mockups.

**Key Features**
- Heuristic evaluation to identify usability issues
- Mood board and style guide
- High-fidelity mockups
- Responsive layouts for desktop and mobile

**Expected Outcome**
Hands-on experience in visual hierarchy, layout design, and responsive design.

## 🔍 1. Heuristic Evaluation

The existing site was evaluated against Jakob Nielsen's usability heuristics.

| Heuristic | Observed Issue |
|---|---|
| Visibility of system status | No visible "Order Online" link despite cart/account features existing on the platform |
| Match between system and real world | Menu and Location pages behave like static content, not an ordering flow |
| Consistency and standards | Main navigation and footer "Quick Links" don't match |
| Error prevention | Several award badges link to placeholder anchors (dead links) |
| Aesthetic and minimalist design | Brand-story paragraph repeated nearly verbatim three times on the homepage |
| Recognition rather than recall | Award badges shown with no captions explaining what they are |
| Flexibility and efficiency of use | No quick path to "Order now" from the hero section |
| Visibility of primary action | "See Menu" and "Location" compete equally — no clear primary CTA |

## 🎨 2. Mood Board — Direction Selection

Three directions were explored:
- **Warm and inviting** *(selected)* — crimson red, cream, and coffee-brown evoking the physical cafe experience
- **Minimal and modern** — muted neutrals with a single crimson accent
- **Bold and youthful** — higher-contrast crimson and gold

"Warm and inviting" was selected to support the brand's "Coffee & Community" positioning while improving visual hierarchy over the current template.

## 🖌️ 3. Style Guide

### Color Palette
| Name | Hex | Usage |
|---|---|---|
| Crimson 600 | `#9E2A2B` | Primary brand color / headers / primary CTA |
| Cream 50 | `#F6EFE4` | Page background |
| Coffee Brown 700 | `#4A3222` | Body text / footer |
| Gold 500 | `#B8863B` | Accents / award & rewards highlights |
| Green 600 | `#5B7F3B` | Success states / order confirmed |

### Typography
- Page title: 30px, medium — serif for warmth
- Section heading: 20px, medium
- Body text: 16px, regular, sans-serif
- Caption / metadata: 13px, regular

### Spacing & Shape
- Base spacing unit: 8px
- Card corner radius: 10px
- Single hero image/video per section (no repeated blocks)

## 🖥️ 4. High-Fidelity Mockups

### Homepage
- Single, clear hero CTA ("Order Online") instead of two competing links
- Brand story consolidated into one section
- Award badges shown with captions and real links (no dead links)

### Navigation
- Unified nav and footer link set: Home, Order Online, Menu, Location, Rewards, History, Career, Contact
- "Order Online" added as a primary nav item

## 📱 5. Responsive Design

- Desktop: 1440px width
- Mobile: 375px width
- Sticky mobile header keeps "Order Online" always visible
- Award badge row becomes a horizontally scrollable carousel on mobile

## ✅ Outcome

This project applies:
- **Visual hierarchy** — through a single dominant CTA and consolidated narrative
- **Layout design** — through unified navigation and captioned content
- **Responsive design** — through the desktop/mobile breakpoint system

directly addressing the usability issues found in the heuristic evaluation of the original site.
